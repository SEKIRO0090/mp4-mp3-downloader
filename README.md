# Mp4 and Mp3 Downloader (Termux)

This is a simple script which allows to download songs amd videos directly into your Download folder instead of using shady websites which can sometimes lead to malicious software being installed on your system (or annoying ads that we all hate!)
Paste link → get file in your normal Download folder. That's it.(simple!)

### What it does
1. Download full video (MP4)
2. Download audio only (MP3)
3. Spotify song downloader (experimental & glitchy rn, works sometimes, working on it!)

Everything saves straight to /storage/emulated/0/Download/

### How to use 
1. In Termux run these once:
```
      python downloader. py
```

  ### Note
  if it says directory not found first mv the script to termux or enter /sdcard/Download/ ( cd /sdcard/Download/ )
  also should've granted termux storage access to download the file into the folder
